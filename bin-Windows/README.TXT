This directory contains a pre-compiled version of "MACE4 2004-C, May 18,
2004" precompiled for Microsoft Windows.  It was compiled with Cygwin
(cygwin.com) on Windows XP by Norm Megill.

To run the example in this directory, cd to this director in a Command Prompt
window and type:

mace4 < nonmodular-oml.in > test.out

The contents of test.out should match the file nonmodular-oml.out
provided.
